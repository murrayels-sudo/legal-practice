Manual conflict of interest search

Enter a prospective client name and/or matter description to check against all existing clients and matters. Conflicts are also checked automatically when you save a new client or matter.

How conflict checking works

The system automatically checks for conflicts when you:

  • Save a new client — checks name against all existing clients
  • Save a new matter — checks client name and matter description against existing records

If a potential conflict is found, you will be shown a warning and can choose to proceed or cancel. A note is automatically added to the client or matter record.

Use the manual search above to check a prospective client before adding them to the system.
